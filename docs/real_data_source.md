# 真实数据来源说明

## 已完成的数据来源接入

本项目已接入高德 Web 服务 API，使用 POI 关键词搜索方式获取南海区相关企业候选数据。

## 接口方式

- 数据来源：高德 Web 服务 API
- 查询方式：POI 关键词搜索
- 区域限制：南海区
- 关键词示例：
  - 大数据
  - 数据服务
  - 数据安全
  - 工业互联网
  - 智慧城市
  - 数据中心
  - 云平台

## 当前实现效果

- 成功生成候选企业 CSV
- 成功导入数据库
- 前端页面已同步展示真实来源数据
- 图表和统计接口已联动更新

## 当前问题

由于 POI 搜索本质上是地点候选检索，结果中存在部分非目标企业或噪声数据，例如洗车站、商场、景区设施等。

## 当前处理策略

- 将 API 返回结果作为候选企业池
- 通过去重逻辑避免重复导入
- 通过人工复核与规则清洗提高数据质量

## 项目价值

本项目已具备“外部真实数据来源接入 → 候选数据生成 → 数据库导入 → 前端展示”的完整链路。
